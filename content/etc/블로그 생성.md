### 블로그 생성 과정
블로그가 있지만 유레카 과정만 따로 모아보기 위해 블로그를 하나 더 만들었습니다. 이번에는 obsidian 내의 blog 폴더에서 push 된 내용이 있으면 자동으로 제 계정에 다른 repository의 내용을 업데이트 하는 스크립트를 github actions로 사용하고 있고, 블로그는 quartz라는 정적 사이트 생성기를 사용하였고 blog 레포가 푸시 될 때마다 cloudflare에서 자동으로 build 하도록 설정해놨습니다.
